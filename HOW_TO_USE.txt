========================================
TROUBLESHOOTING
========================================

Q: Controls not working?
A: Check console for JavaScript errors
   Verify handleManualMove function exists

Q: Dashboard not showing?
A: Check screen width
   Verify bag array is defined
   Check browser console

Q: Maze too small?
A: Adjust sizing in MazeGame.tsx line 1438:
   width: 'min(50vw, calc(100vh - 200px))'

Q: Layout broken on wide screens?
A: Portrait version is optimized for vertical
   Use standard version for horizontal displays

Q: Text too small?
A: Adjust font sizes in dashboard section
   Lines 1528, 1532, 1538, 1552

========================================
CUSTOMIZATION
========================================

To Change Control Position:
1. Edit MazeGame.tsx line 1394
2. Swap LEFT and RIGHT side sections
3. Controls will move to right side
4. Dashboard will move to left side

To Change Maze Size:
1. Edit MazeGame.tsx line 1438-1440
2. Adjust min() values
3. Increase/decrease maxWidth/maxHeight

To Change Dashboard Width:
1. Edit MazeGame.tsx line 1514
2. Change w-[120px] to desired width
3. Adjust inventory grid if needed
